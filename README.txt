Project Management System (Tkinter + PostgreSQL)
-------------------------------------------------
This is a Project Management System using a PostgreSQL database for storage.

How to Run:
1. Install required package:
   pip install psycopg2

2. Make sure PostgreSQL server is running and you have created:
   - A database named 'project_management'
   - A table named 'projects' with correct structure.

3. Update your PostgreSQL credentials in the script.

4. Run the application:
   python project_management_system_postgres.py

Features:
- Add new project records
- View all saved projects
- Save data into a PostgreSQL database
